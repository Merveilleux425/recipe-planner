"# recipe-planner" 
Recipe Scaler Web Application
A simple and elegant web application to save and scale recipes. Built with Node.js, Express, and vanilla JavaScript.

Features
Create and save new recipes with ingredients.
View a list of all saved recipes.
Dynamically scale any recipe to a different number of servings.
Fully responsive design for mobile and desktop.
Prerequisites
Before you begin, ensure you have the following installed on your machine:

Node.js (v14 or later recommended)
npm (usually comes with Node.js)
Installation
Follow these steps to set up the project locally:

Clone the repository:Open your terminal or command prompt and run:
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
Navigate to the project directory:
cd YOUR_REPOSITORY_NAME
Install dependencies:This command will install Express and any other required packages listed in package.json.
npm install
Running the Application
Start the server:In the same terminal, run the following command:
npm start
You should see the message: Server is running on http://localhost:3000.
Open the application:Open your web browser and navigate to:http://localhost:3000
Usage
Save a Recipe: Fill out the form at the top of the page with a recipe name, original servings, and ingredients. Click "Save Recipe".
Scale a Recipe: Find a recipe in the list below. Enter a new number of servings in the "Scale Recipe" input box and click the "Scale" button. The ingredient quantities will update automatically.
